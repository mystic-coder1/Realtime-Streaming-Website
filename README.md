# 📺 Realtime Streaming Website

A full-stack **real-time streaming platform** that enables users to broadcast and watch live video streams seamlessly. Built with modern web technologies, this project demonstrates real-time communication, media streaming, and scalable backend architecture.

---

## 🚀 Features

- 🔴 **Live Video Streaming**
  - Broadcast real-time video to multiple users
- 👥 **Multiple Viewers Support**
  - Concurrent viewers can join streams instantly
- 💬 **Real-time Interaction**
  - Chat or communication during live streams
- 🔐 **User Authentication**
  - Secure login/signup system
- ⚡ **Low Latency Streaming**
  - Fast and smooth video delivery
- 📱 **Responsive UI**
  - Works across desktop and mobile devices

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/mystic-coder1/Realtime-Streaming-Website.git
cd Realtime-Streaming-Website
```

### 2️⃣ Install dependencies
```bash
npm install
```

### 3️⃣ Start the server
```bash
npm start
```

### 4️⃣ Open in browser
```
http://localhost:3000
```

---

## 🧠 How It Works

- Uses **real-time protocols** to establish peer-to-peer communication  
- Backend manages:
  - User sessions
  - Stream signaling
- Frontend handles:
  - Video capture
  - Stream rendering  
- **Socket connections** sync data instantly between users
